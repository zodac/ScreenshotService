post

Creates a new screenshot request with a list of URLs. This is a blocking request and will return once all screenshots have been saved.

post /

Creates a new screenshot request with a list of URLs. This is a blocking request and will return once all screenshots have been saved.

  • Request
  • Response

Body

Type: application/json

Example:

{
  "urls": [
    "http://www.url1.ie",
	  "http://www.url2.com"
  ]
}

HTTP status code 201

The screenshot request has been accepted and processed, with a 'location' header directing to the result

HTTP status code 400

The screenshot request has been rejected due to an invalid URL in the request

Body

Type: application/json

Example:

{
  "invalidUrls": [
    "invalid-url"
  ]
}

/{jobId} get

get

Retrieves a single job ID and its screenshot file names

get /{jobId}

Retrieves a single job ID and its screenshot file names

  • Request
  • Response

URI Parameters

  • jobId: required (string)

HTTP status code 200

A single job ID result

Body

Type: application/json

Example:

{
  "jobId": 1,
  "screenshotFileNames": [
    "2019-37-29_01:37:28_1.png",
    "2019-37-29_01:37:28_2.png",
    "2019-37-29_01:37:28_3.png",
    "2019-37-29_01:37:28_4.png"
  ]
}

HTTP status code 404

No job ID exists for the supplied ID
